# School Management System

A premium bilingual (Arabic/English) Saudi School Management System built with React, Vite, Tailwind CSS, and Recharts.

## Features
- 7 roles: Principal, Admin, Teacher, Student, Parent, Accountant, Transport Manager
- 18 pages with full Saudi dummy data (Arabic names, SAR amounts, Hijri/Gregorian dates)
- 3 branches: Riyadh Main, Jeddah Branch, Dammam Campus
